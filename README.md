# 🚀 Full Stack Todo Application with CI/CD Pipeline

A complete full-stack Todo application with automated CI/CD pipeline using GitHub Actions.

## 📋 Project Overview

This is a production-ready Todo application built with modern technologies and fully automated CI/CD pipeline. Every push to the main branch automatically triggers testing and build processes.

## 🛠️ Tech Stack

| Layer | Technology |
|-------|------------|
| Frontend | React.js |
| Backend | Node.js, Express.js |
| Database | MongoDB |
| CI/CD | GitHub Actions |
| Container | Docker |

## ✨ Features

- Create, Read, Update, Delete (CRUD) todos
- Mark todos as complete/incomplete
- Real-time updates
- Data persistence with MongoDB
- RESTful API architecture
- Docker containerization
- Automated CI/CD pipeline
